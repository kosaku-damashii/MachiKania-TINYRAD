# MachiKania-TINYRAD
<img width="1926" height="1524" alt="image" src="https://github.com/user-attachments/assets/96b05c61-442f-429d-8007-1b1b49b90dad" />
FMラジオモジュールRRD-102V2.0（RDA5807M搭載）を使ったFMラジオ。<BR>
MachiKania type P/PU のGROVE(I2C)端子に繋げて使う。<BR>
<BR>
配線図<BR>
MachiKania type P開発基板上にI2Cのプルアップ抵抗が付いている前提。<BR>
L-OUTとR-OUTがモジュールの説明書と違うと思うが、ICのデータシートを正とするならばこうなるはず（テスターで確認済）<BR>
アンテナには長めのジャンパー線とか付ければ良い。<BR>
<img width="1515" height="425" alt="image" src="https://github.com/user-attachments/assets/d74703ed-c11a-47fb-9d52-8367c3ef9441" />
<BR>
操作方法<BR>
MachiKania type P/PU本体の上下キーで音量調整<BR>
STARTキーで放送局をシーク<BR>
